## 생각하기
  > 다이나믹 프로그래밍
  > 수학

문제를 읽고 예시들을 생각해보면 수학적으로 접근해서 풀 수 있다.

1. 2 X 4
 -> 7번으로 1X1로 쪼갤 수 있다.

2. 3 X 4
 -> 11번으로 1X1로 쪼갤 수 있다.

 => 종합해보면 N * M - 1로 풀 수 있다.


## [2163 풀이]

```c++
#include <iostream>
using namespace std;
int N, M;

int main() {
	cin >> N >> M;

	cout << N * M - 1 << "\n";
}
```
